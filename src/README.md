# IoCameraCapture

基于海康 `HCNetSDK` 的 WinForms 工具，用于设备登录、IO 报警抓图、图片存储和触发测试。

## 页面区域

- `通讯参数`：设备 IP、端口、用户名、密码、通讯号码、SDK 目录
- `IO 接入参数`：IO 类型、IO 型号、输入号、抓图通道、去抖时间、脉冲时间、图片规格、存图位置
- `触发测试`：内部触发、外部触发测试、触发次数、触发间隔
- `最新抓图预览`：显示最近一次保存的图片
- `运行日志`：显示登录、布防、触发、抓图和错误信息

## 内置 IO 类型适配

- `干接点常开`
- `干接点常闭`
- `NPN 开集电极`
- `PNP 开集电极`
- `TTL 5V 上升沿`
- `TTL 5V 下降沿`
- `继电器脉冲 12/24V`
- `自定义 / 厂商专用`

每种类型都包含：

- 接线说明
- 有效电平说明
- 触发沿说明
- 默认去抖时间
- 默认脉冲时间

当前已用于：

- 页面兼容类型预设
- 真实外部触发与模拟外部触发的去抖过滤
- 触发测试时的日志标识
